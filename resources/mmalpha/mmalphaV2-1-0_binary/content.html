The content of this directory is described in this README file.

From this page, you can access what follows:

  • The binary directories for windows, Linux and Mac OS X. These directories contain the binary files for Domlib, Polylib, Pip etc.
  • The configuration directory. It contains information regarding the configuration of MMAlpha.
  • The Mathematica package lib directory. It contains the Mathematica files of MMAlpha.
  • The demonstration directory. It contains many examples of MMAlpha usage.
  • The documentation directory.
  • The source directory. It contains the source files of Domlib, Polylib etc.
  • The lib directory. It contains the source files (Mathematica packages) of MMAlpha.
  • The tests directory. It contains the test files for all packages.
  • The maintenance directory. This directory, which may not be accessible on your distribution, contains files concerned with the maintenance of MMAlpha.
