Demonstrations of MMAlpha

Up! (welcome page)

This directory contains demonstration notebooks. All these notebooks are accessible from the master notebook, which can be open once MMAlpha has been started, using the "start[]" command.

The following demos are available (check the master notebook to see what demos are actually running...):

  • Getting started
  • Introduction
  • A delay line
  • The Domlib
  • A Fifo
  • A Fir filter
  • The matrix vector multiplication
  • The matrix multiplication
  • The Samba architecture
  • Scheduling
  • Structured scheduling
