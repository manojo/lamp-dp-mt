Test directory

up (welcome page)

This directory contains test files for MMAlpha. Each package has a test directory, which contains test files. These tests can be executed from MMAlpha, by evaluating the command tests["package"], where package is the name of the package.

In addition, this directory contains a notebook called tests.nb which contains information regarding tests.
