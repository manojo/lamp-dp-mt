MMAlpha Documentation







This page contains links to the documentation documents of MMAlpha, these documents are classified into three categories: general documentation, Alpha langage documentation, MMAlpha environment documentation and documentation for MMAlpha developpers.
All the other documentations are gathered here.

Here is a schematic of the file hierarchy of the MMAlpha software

  • General documentation:
    • The MMAlpha QuickStart: should be read first (25 pages).
    • The MMAlpha tutorial: overview of the whole system (100 pages)
    • The MMAlpha reference manual: contains the description of all the functions available in MMAlpha (60 pages) 

  • Alpha Langage
    • The Syntax of Alpha.
    • The Alpha language (22 pages, Reasearch report number 827). This document contains a rather formal description of the Alpha language. Another good introduction is Christophe Mauras thesis.
    • Structuration of the Alpha Language (7 pages). Describes the use construct and details the inlining procedure. Published in MPPM95.


  • MMAlpha environment
    • make sure to read the  MMAlpha QuickStart,  MMAlpha tutorial, and  MMAlpha reference manual mentionned above before getting these new documentations
    • Regular array synthesis using Alpha (14 pages). This document explains the philosophy of Alpha. Unfortunately, the commands shown in the paper do not always work because it is quite old..
    • Hierarchical Static Analysis of Structured Systems of Affine Recurrence Equations (10 pages). A paper about static analysis with parameters and subsystems, published in ASAP96.
    • Loop nest synthesis using the polyhedral library (8 pages). This document is rather technical, and explains how one can enumerate the points of a convex polyhedron.
    • Code generation from Alpha (18 pages).


  • For MMAlpha developpers:
    • The Polyhedral library: Irisa technical report n° 785, describes operations and implementation of the polyhedral library.
    • The Syntax of the Abstract Syntax Tree (AST) of MMAlpha.
    • How to develop and integrate new functions in MMAlpha
    • Code generation from Alpha (18 pages).
    • The Alpha compiler and Uncompiler (technical note 901) about the parsers.

  • Not available from internet:
    • Structuration of the Alpha Language (7 pages). Describes the use construct and details the inlining procedure. Published in MPPM95.
    • Structured Systems of Affine Recurrence Equations and their Applications (22 pages). Basically a (shorter) english version of Florent's thesis. Submitted to Toplas in sept 97.
    • Libraries of Schedule-Free Operators in Alpha (10 pages). A paper about the first experiments on binary expansion in Alpha (replacing operators on real scalar types with subsystems implementing them at the bit level). Published in ASAP'97.
    • Documentation of Each packages can be found in $MMALPHA/doc/Packages (but can only by view by Mathematica 3.0)

      • If you really think that you need something else, you can browse the $MMALPHA/doc/ directory (it is alittle messy)
