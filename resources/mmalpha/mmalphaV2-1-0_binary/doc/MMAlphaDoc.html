MMAlpha

Here are a few publications or technical notes regarding MMAlpha. Some of them are only of historical value.
Make sure you read the general documentation before opening one of the following documents, as they are more specialized.
  • Regular array synthesis using Alpha (14 pages), explains the philosophy of Alpha. Unfortunately, the commands shown in the paper correspond to an earlier version of the software...
  • Hierarchical Static Analysis of Structured Systems of Affine Recurrence Equations, 10 pages, is a paper about static analysis with parameters and subsystems, published in ASAP96.
  • Loop nest synthesis using the polyhedral library, 8 pages, rather technical, explains how one can enumerate the points of a convex polyhedron.
  • Code generation from Alpha, 18 pages, explains how C code is generated from an Alpha program.
