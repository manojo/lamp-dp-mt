This documentation gives conventions regarding the development of packages. A reference manual is being developped. A document describing how to compile Domlib is being prepared here.

Other documents (that may be partially obsolete) are:

  • Irisa technical report nr 785 describes the operation and the implementation of the polyhedral library, the "engine" of MMAlpha
  • The Syntax of the Abstract Syntax Tree (AST) of MMAlpha is described in this man-like document.
  • The Alpha Compiler and Uncompiler (technical note 901) describes the parsers of MMAlpha.
  • Structured Systems of Affine Recurrence Equations and their Applications, 22 pages, is an english summary of Florent Dupont de Dinechin's thesis.
  • Libraries of Schedule-Free Operators in Alpha, 10 pages, is a paper about the first experiments on binary expansion in Alpha (replacing operators on real scalar types with subsystems implementing them at the bit level). It was published in ASAP'97.
  • A documentation for the main MMAlpha packages can be found in $MMALPHA/doc/Packages as Mathematica Notebooks.
  • If you really think that you need something else, you can browse the $MMALPHA/doc/ directory (it is a little messy...)
