In this directory, you will find one subdirectory of each one of (allmost) all packages of MMAlpha. Each one of this directory contains a notebook which present a package.

For example, there is a package named Alpha. Its documentation is in the Alpha directory, in a notebook named Alpha.nb.

You can start any such notebook in any notebook by evaluating the expression docLink["Alpha"], which will paste an active button in your notebook. Clicking on this button opens the Alpha.nb notebook.
